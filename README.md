# Epic Parse
## Project Overview
As a long-time World of Warcraft player, I understand how frustrating it can be when your favorite class or specialization is underperforming. Epic Parse explores how class balance and other vital aspects affect player satisfaction. Balancing classes is essential for keeping players happy and engaged. By analyzing player feedback, this project aims to provide insights that can contribute to WoW's success.
